<h2>
  简介：
</h2>
<p>
    这是一个面向在校学生以及学生社团的网站。
    对于社团用户，他们可以在这个网站上发布自己即将举办的活动，并附上图文并茂的介绍。
    对于学生用户，他们可以在主页上看到正在举办的活动、即将举办的活动以及热门的活动，并且可以看到大家对活动的评论，以及社团之间的排行榜。
    
</p>
